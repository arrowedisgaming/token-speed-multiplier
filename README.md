## What it does

Adds a single slider to your module settings: **Token Animation Speed Multiplier.** Slide it up and your tokens fly across the canvas. Slide it down and they crawl.

- `1` is Foundry's default speed.
- `2` makes tokens move twice as fast.
- `0.5` makes them move half as fast.
- Anywhere from `0.1` to `10`, in steps of `0.1`.

The change takes effect the next time a token moves.

## What it doesn't do

This is a **purely visual** tweak. 

You can play D&D 5e, Black Flag, Delta Green, Pathfinder, or anything else — the module doesn't care. It only changes how fast tokens slide across the screen.

## Use

1. Go to **Configure Settings → Module Settings → Token Speed Multiplier**.
2. Drag the **Token Animation Speed Multiplier** slider to whatever feels good.
3. Save. Move a token. Adjust to taste.

## Compatibility

- **Foundry VTT:** v14 minimum, v14 verified.
- **Systems:** any
